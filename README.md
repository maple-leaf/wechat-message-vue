# wx-messager-vue

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

## 微信信息组件

实现思路见 react 组件的 [README](https://github.com/maple-leaf/wechat-message-react/blob/master/README.md)。

差异只是自定义 `render`，`messagePrefix` 和 `messagePostfix` 在 `vue` 这使用的是 `slot` 来实现。
